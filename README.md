# Addidas-Sales-Analysis

## Project Overview
This project is an interactive Excel-based sales dashboard created using Adidas sales data. It provides a clear overview of sales performance and profitability to support data-driven decision-making.

## Objective
The objective of this dashboard is to track key sales and profit indicators, identify trends, and highlight areas of strong and weak performance.

## Tools Used
- Microsoft Excel

## Key Metrics / KPIs
- Total Sales: Total revenue generated from sales  
- Total Units Sold: Number of products sold  
- Total Profit: Net profit after costs  
- Average price per Unit: Average revenue generated per unit sold  
- Average Profit Margin: Average profit as a percentage of sales  

## Dashboard Features
- KPI cards summarizing key performance indicators  
- Interactive charts and visuals  
- Filters for better data exploration  
- Clean and user-friendly layout

## Insights
- Highlights overall sales and profitability performance  
- Shows trends in revenue and units sold over time  
- Identifies high-performing and low-performing sales periods

## How to Use
1. Download the Excel file from this repository  
2. Open it using Microsoft Excel  
3. Use the filters and visuals to explore sales performance  
4. Review the KPIs to understand revenue, volume, and profit trends
